## Описание 📜

Данный проект представляет собой пример моей работы, выполненной в компании **Aezakmi Group**. Приложение разработано с использованием **SwiftUI** и позволяет создавать видео на основе фотографий и текстовых описаний благодаря интеграции с **MiniMax API**.

В проекте также реализована интеграция **Apphud** для управления подписками, настройки и отображения **paywall**-экранов, а также сбора аналитики. Для удобства разработки и тестирования предусмотрен моковый режим работы, позволяющий проверять функциональность без выполнения реальных запросов к API и расходования токенов.
